# Routes

Below is a list of all available routes in the application:

## Homepage

* **GET** `/`: Render the homepage

## About

* **GET** `/about`: Render the about page

## Projects

* **GET** `/projects`: Render the projects page

## Blog

* **GET** `/blog`: Render the blog homepage
* **GET** `/blog/:category`: Render the blog posts for the given category
* **GET** `/blog/:category/:tag`: Render the blog posts for the given category and tag
* **GET** `/blog/:category/:tag/:page`: Render the blog posts for the given category, tag, and page number

## Snippets

* **GET** `/snippets`: Render the snippets page

## Coding

* **GET** `/coding`: Render the coding page

## Ask AI

* **GET** `/ask-ai`: Render the ask AI page

## Quick Links

* **GET** `/quick-links`: Render the quick links page

## Games

* **GET** `/games`: Render the games page

## Resources

* **GET** `/resources`: Render the resources page
